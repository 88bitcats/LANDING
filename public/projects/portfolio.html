P.T. (for prototype) material design themed podcasting prototype for signl.fm using primarily google api and polymer components.

Mashup exploring Leap Motion's capabilities. The app moves through Google Streetview using Leap Motion controls, while implementing markers from Translink, Google Places, Twitter, and Instagram. The app uses Bootstrap on the frontend, with Node.js and Hapi server on the background.
